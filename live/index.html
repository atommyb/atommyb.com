Prototypes & Demos

Here are a few prototypes made in conjunction with a couple of code legends at Rockpool - Tom Haigh & Lee Howarth

Jukebox powered by YouTube

A Social Jukebox, powered by Youtube, Soundcloud & Spotify

  • Add your favourite sources from Youtube, Vimeo, Soundcloud
  • Spotify support added, with local install & desktop app server (Tom H!)
  • Open Graph data powered
  • Vote your favourites up the queue
  • Perfect for the office (or creating arguments in it!)
  • Node.js for server
  • Express for UI hookup
  • Websockets for sync
  • CSS3 Transitions, Transforms for sexiness
  • Source on GITHub

Pingo - A Pictionary™-like game

A digital mobile & desktop version of a classic drawing game

  • Try it on your mobile and your desktop
  • Canvas API (custom library by Tom Haigh)
  • Knockout.js MVVM
  • Node.js backend
  • Websockets for syncing
  • HTML5 for the win
  • CSS3 Transitions, Transforms
  • Source on GITHub

IPTV 2nd Screen Remote Control

A prototype 2nd screen experience & remote control for a fictional IPTV service.

  • Try is on your mobile and your desktop
  • Remote controlled via mobile Safari, Chrome, Firefox
  • Websockets (.NET server work by BlueGhost (Lee) )
  • Windows Azure hosted
  • HTML5 Local Storage
  • CSS3 Transitions, Transforms
  • Source (private for now)

Interactive Distributed Presentation

HTML5 presentation slides on your desktop or iPad, controlled from your mobile. Distributed viewing by others joining URL & slide synching across multiple browsers using websockets

  • Try it on your mobile and your desktop
  • Remote controlled via Safari on iOS (Chrome probably works)
  • Node.js
  • Websockets (nodejitsu hosted)
  • HTML5
  • CSS3 Transitions, Transforms
  • Inspired by Impress.js presentations
  • Source (poorly organised) on GITHub
